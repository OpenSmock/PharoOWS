# PharoOWS

OGC Web Services support for Pharo.

## Getting Started

### Installing PharoOWS

Pharo 11 :

```smalltalk
Metacello new
   baseline: 'PharoOWS';
   repository: 'github://OpenSmock/PharoOWS:main';
   load.
```

### Dependencies

PharoOWS loads these others projects :
  - [XMLParser](https://github.com/pharo-contributions/XML-XMLParser)
